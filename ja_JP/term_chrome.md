# 用語：Google Chrome

## 説明

世界的なWeb検索エンジン、Web広告大手の米グーグル（Google）社が開発・公開しているWebブラウザ。

無償配布されており、誰でも自由に入手・使用することができる。

Windows版、macOS（Apple Mac OS X）版、Linux版、iOS版、Android版が用意されている。
